# CDNA FIX5.0 SP2 Custom Application Messages

## Introduction
This project builds custom application messages for QuickFIX/J according to the CDNA FIX5.0 SP2 implementation.

The QuickFIX dictionary is located in the `src/main/resources/messages` directory.

## References
See [quickfixj-custom-orchestration-examples](https://github.com/quickfix-j/quickfixj-custom-orchestration-examples) 
for descriptions of how to customise QuickFIX/J message packages and examples of how to use them in a project.

See [simple-examples-spring-3](https://github.com/NadexWeb/simple-examples-spring-3)
for examples of using this distribution in a Spring 3 application.
